## 安装
npm install -D vitepress
## 安装向导
 npx vitepress init
  - 接下来是一些配置信息，根据自身喜欢选择即可

## 目录结构
 ```
 .
├─ docs -- 根目录
│  ├─ .vitepress // 配置文件、开发服务缓存、构建、主题
│  │  └─ config.js
│  ├─ api-examples.md
│  ├─ markdown-examples.md
│  └─ index.md
└─ package.json
 ```
